新建或编辑文件
# vim filname
参数说明：
:wq 保存后退出vi，若为 :wq! 则为强制储存后退出（常用）
:w 保存但不退出（常用）
:q 退出（常用）
:w! 若文件属性为『只读』时，强制写入该档案
:q! 放弃修改并退出
:e! 将档案还原到最原始的状态！
按键说明：
i 进入编辑模式
Esc 退出编辑模式
u 一次，撤销上一步的操作；两次，撤销所有的操作

显示主机名
# hostname

修改主机名
# vim /etc/sysconfig/network
HOSTNAME=your hostname
# vim /etc/hosts
127.0.0.1 localhost your hostname
# shutdown -r now

查看主机网络配置
# ifconfig

防火墙:
# service iptables status --查看防火墙状态
# service iptables stop  --停止防火墙
# service iptables start  --启动防火墙
# service iptables restart  --重启防火墙
# chkconfig iptables off  --永久关闭防火墙
# chkconfig iptables on  --永久关闭后启用

关机：
# shutdown -h now  --立即关机
# shutdown -h 12:30  --设置系统在12:30关机
# shutdown -r now  --立即重启
# shutdown -r +2  --设置系统在2分钟后重新启动
# shutdown -c  --取消

查看系统是32位还是64位：
# uname -a
Linux hadoop1 2.6.32-504.el6.i686 #1 SMP Wed Oct 15 03:02:07 UTC 2014 i686 i686 i386 GNU/Linux
# uname -m
i686
# arch
i686
x86_64：64位系统
i686、i386：32位系统。i686只是i386的一个子集
